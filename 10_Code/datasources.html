The data used for this project are originating from many different sources which are described here.

Migration data

The migration data is from the UN and its report International migration flows to and from selected countries: The 2015 revision.
The aggregated country data can be found here. The migration flows between country is based on this data. In many cases there have been many data points per country and year. This resulted in the following rule set of selecting the country data:

Coverage (foreigners vs. citizens vs both)
  • In general both have been selected
  • For Canada, US & New Zealand both is not availabe, foreigners selected
Criteria (Citizenship vs. place of birth vs. residence)
  • For most countries only the criteria residence was left after selecting both as coverage
  • For France citizenship and residence have been left but only residence had data for immigrants and emigrants therefore his has been selected
  • The US have only data for the criteria place of birth
  • Canada has only data for the criteria citizenship
  • For New Zealand residence has been selected
For more details about the selection criteria and where the respective data is coming from, please have a look in the documentation from the UN.

Population data

The population data was taken from the worldbank.

Countries and continents

The allocation of countries to continents is based on worlddata.info.

Centroids of countries

The centroids of the countries which have been used to draw the arrows in the map are coming from gothos.info.
